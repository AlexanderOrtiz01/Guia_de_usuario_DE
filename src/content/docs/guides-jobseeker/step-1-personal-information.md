---
title: "Paso 1: Información Personal"
description: Guía para completar la información personal
---

En este primer paso del registro, completarás tu información personal básica. Sigue estos pasos:

## Paso 1: Datos Básicos

Completa los siguientes campos:
- **Nombres** (First name y Last name)
- **Código de país** y **número de teléfono**
- **Foto de avatar**: Haz clic en "Change" para subir una imagen de perfil
- **Nivel de educación** (Education level): Selecciona el nivel más alto completado

![Información personal básica](https://jam.dev/cdn-cgi/image/width=2560,quality=100,dpr=1.25/https://cdn-jam-screenshots.jam.dev/80141684180e876e9ee7e8967db13575/screenshot/049ead59-d058-45db-b9ee-4d086656f41c.png)

## Paso 2: Dirección y Profesión

Ingresa tu información de ubicación y profesional:
- **País** (Country)
- **Estado** (State)
- **Calle** (Street)
- **Ciudad** (City)
- **Código postal** (ZIP / Postal code)
- **Industria** (Industry): Puedes seleccionar hasta 2 industrias
- **Profesión** (Profession): Puedes seleccionar hasta 2 profesiones

![Dirección y profesión](https://jam.dev/cdn-cgi/image/width=2560,quality=100,dpr=1.25/https://cdn-jam-screenshots.jam.dev/469fc4bb84ac113b43df124ecae2db56/screenshot/a1a0d574-09e5-4489-aedd-cc6767420147.png)

## Paso 3: Idiomas y Nivel de Experiencia

Completa la información adicional:
- **Idiomas** (What languages do you speak?): Puedes seleccionar más de un idioma con su nivel de competencia
- **Nivel de experiencia** (Experience Level): Indica tu experiencia profesional

**Opciones de navegación:**
- **Save for later**: Si deseas guardar tu progreso y continuar más tarde
- **Next**: Si toda la información está correcta, presiona este botón para continuar al siguiente paso

![Idiomas y nivel de experiencia](https://jam.dev/cdn-cgi/image/width=2560,quality=100,dpr=1.25/https://cdn-jam-screenshots.jam.dev/ad3ec01fe8481ffc529d79a1f9493c77/screenshot/c217a163-c73c-4047-86a0-6f578f1daed6.png)

## Descripción De Elementos

| Número | Campo | Descripción | Obligatorio |
|--------|-------|-------------|-------------|
| 1 | First name y Last name | Campos para ingresar el nombre y apellido | Sí |
| 2 | Phone number | Campo para ingresar el número de teléfono | No |
| 3 | Avatar | Botón para subir o cambiar la foto de perfil | No |
| 4 | Education level | Nivel más alto de educación completado | Sí |
| 5 | Country | Selección del país de residencia | Sí |
| 6 | State | Selección del estado/provincia | Sí |
| 7 | Street | Dirección de calle | Sí |
| 8 | City | Ciudad de residencia | Sí |
| 9 | Industry | Selección de hasta 2 industrias de interés | Sí |
| 10 | Profession | Selección de hasta 2 profesiones | Sí |
| 11 | ZIP / Postal code | Código postal de la dirección | Sí |
| 12 | What languages do you speak? | Selección de idiomas con nivel de competencia | Sí |
| 13 | Experience Level | Nivel de experiencia profesional | Sí |
| 14 | Save for later | Botón para guardar y continuar después | N/A |
| 15 | Next | Botón para avanzar al siguiente paso | N/A |
