---
title: Codigo De Verificacion
description: Guía para el código de verificación
---

Después de registrarte, deberás verificar tu correo electrónico ingresando un código de verificación que recibirás en tu email. Sigue estos pasos:

## Paso 1: Pantalla de Verificación

Una vez que hayas completado el registro, aparecerá un cuadro solicitando el código de verificación de email.

![Pantalla de verificación de email](https://jam.dev/cdn-cgi/image/width=2560,quality=100,dpr=1.25/https://cdn-jam-screenshots.jam.dev/e7e1acc2ffeb3f02e384f9eef41d1f29/screenshot/078f079f-42ea-4065-9737-c36d91705e9e.png)

## Paso 2: Revisar Bandeja de Entrada

Abre la bandeja de entrada del correo electrónico que registraste. Encontrarás un mensaje con el código de verificación. Copia ese código.

![Código de verificación en el email](https://jam.dev/cdn-cgi/image/width=2560,quality=100,dpr=1.25/https://cdn-jam-screenshots.jam.dev/37744e69cd59221a7bc98ca5239a2938/screenshot/6c130da8-a9b8-405d-9f1c-27174e6ac7a4.png)

## Paso 3: Ingresar el Código

Regresa a la pantalla de verificación y pega el código en el campo correspondiente. Luego presiona el botón para confirmar.

![Ingresando el código de verificación](https://jam.dev/cdn-cgi/image/width=2560,quality=100,dpr=1.25/https://cdn-jam-screenshots.jam.dev/52a8c8c28a915f422643555fe6f5ce78/screenshot/452b79cd-da92-4af8-9957-47fe69d17f7a.png)

## Paso 4: Confirmar Verificación

Presiona el botón para confirmar la verificación del código.

![Botón de confirmación](https://jam.dev/cdn-cgi/image/width=2560,quality=100,dpr=1.25/https://cdn-jam-screenshots.jam.dev/9035f0ec286dd7cae4992ee0e48ce615/screenshot/9db00be0-aa8b-4ecd-8841-95a83b064114.png)

Una vez verificado el código, podrás continuar con el proceso de configuración de tu perfil.
