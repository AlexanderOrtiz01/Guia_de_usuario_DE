## Descripción De Elementos

| Numero De Seccion | Descripcion |
|-------------------|-------------|
| 1. Lista de chats recientes | Muestra conversaciones activas con nombres e íconos. |
| 2. Encabezado del chat abierto | Indica el nombre y miembros del grupo actual. |
| 3. Navegación: Chats | Botón que indica que estás en la sección de mensajería. |
| 4. Navegación: Usuarios | Acceso a la gestión de usuarios del sistema. |
| 5. Navegación: Grupos | Permite gestionar o crear grupos de usuarios. |
| 6. Adjuntar archivo | Botón para enviar documentos o archivos. |
| 7. Grabar audio | Botón para grabar y enviar mensajes de voz. |
| 8. Emoji | Acceso rápido a emoticones para los mensajes. |
| 9. Campo de texto | Área para escribir el mensaje antes de enviarlo. |
| 10. Enviar mensaje | Botón para enviar el mensaje escrito. |
