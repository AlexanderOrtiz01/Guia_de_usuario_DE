## Pasos Para Enviar Datos Adjuntos

### Paso 1: Presionar El Botón De Adjuntar

![Botón de adjuntar archivos](https://jam.dev/cdn-cgi/image/width=2560,quality=100,dpr=1.25/https://cdn-jam-screenshots.jam.dev/b75e299b2c4b90c1883b604dbff3c1be/screenshot/86186146-05f2-4641-a2ed-c1d36cb8cbf4.png)

1. Abre la conversación donde deseas enviar un archivo adjunto
2. En la parte inferior del chat, localiza el icono de **clip** o **adjuntar**
3. Haz clic en el icono para abrir las opciones de archivos adjuntos
4. El sistema mostrará los diferentes tipos de datos que puedes enviar

### Paso 2: Elegir El Tipo De Dato A Enviar

![Opciones de tipos de archivos](https://jam.dev/cdn-cgi/image/width=2560,quality=100,dpr=1.25/https://cdn-jam-screenshots.jam.dev/fb32cdf054ec801ba9b621f1240d51f1/screenshot/126882a4-2809-48ad-869e-7d0d74c5eaad.png)

Una vez presionado el botón de adjuntar, podrás elegir entre diferentes tipos de archivos:

| Tipo De Archivo | Descripción |
|-----------------|-------------|
| **Adjuntar imagen** | Permite enviar archivos de imagen desde tu dispositivo en formatos como JPG, PNG, GIF, WEBP, etc. |
| **Adjuntar video** | Envía archivos de video en formatos como MP4, AVI, MOV, MKV y otros formatos compatibles |
| **Adjuntar audio** | Adjunta archivos de audio previamente grabados en formatos como MP3, WAV, OGG, etc. |
| **Adjuntar archivo** | Permite adjuntar cualquier tipo de documento como PDF, Word, Excel, PowerPoint, ZIP y otros formatos |

### Envío Del Archivo

1. Selecciona el tipo de archivo que deseas enviar
2. Busca y selecciona el archivo específico de tu dispositivo
3. Una vez seleccionado, el archivo se adjuntará automáticamente al chat
4. Opcionalmente, puedes agregar un mensaje de texto junto con el archivo
5. Haz clic en el botón de **enviar** para compartir el archivo adjunto
