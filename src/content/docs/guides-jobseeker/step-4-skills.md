title: "Paso 4: Habilidades"
description: Guía para completar las habilidades
---

En este paso, agregarás tus habilidades técnicas y blandas. La plataforma ofrece sugerencias basadas en tu perfil para facilitar el proceso.

## Paso 1: Seleccionar Habilidades

Algunas habilidades son sugerencias de IA generadas en base al perfil que has creado hasta el momento. También puedes buscar manualmente tus habilidades técnicas (Technical Skills) y habilidades blandas (Soft Skills).

![Selección de habilidades](https://jam.dev/cdn-cgi/image/width=2560,quality=100,dpr=1.25/https://cdn-jam-screenshots.jam.dev/6648efd29de1d19880293c8a8c8a0702/screenshot/15971e39-47e6-4da6-aba5-95f138e0679a.png)

## Paso 2: Indicar Nivel de Competencia

Para cada habilidad seleccionada, indica el nivel de competencia que tienes con esa habilidad (Beginner, Intermediate, Advanced, Expert).

![Nivel de competencia](https://jam.dev/cdn-cgi/image/width=2560,quality=100,dpr=1.25/https://cdn-jam-screenshots.jam.dev/9a02133a2b2159b5f978effbfce1ef77/screenshot/37c63c2e-c61b-46d5-a3d8-52e5ccd9761c.png)

## Paso 3: Continuar al Siguiente Paso

Presiona el botón **Next** para avanzar al paso final.

![Botón Next](https://jam.dev/cdn-cgi/image/width=2560,quality=100,dpr=1.25/https://cdn-jam-screenshots.jam.dev/744479a2d8f940a10e7f11d4770605bb/screenshot/9aa15c39-5b03-44fc-8352-593452f98d56.png)

## Descripción De Elementos

| Número | Campo | Descripción | Obligatorio |
|--------|-------|-------------|-------------|
| 1 | Technical Skills | Habilidades técnicas para agregar al perfil | Sí |
| 2 | Soft Skills | Habilidades blandas para agregar al perfil | Sí |
| 3 | Skill Level | Estrellas de nivel de competencia en esa habilidad | N/A |
| 4 | Next | Botón para avanzar al siguiente paso | N/A |
