## Pasos Para Crear Un Chat Con Usuario

### Paso 1: Acceder A La Sección De Usuarios

![Pantalla principal de chat](https://jam.dev/cdn-cgi/image/width=2560,quality=100,dpr=1.25/https://cdn-jam-screenshots.jam.dev/912ef6ff80b7c8b0143924fc92fea927/screenshot/0789bd24-c1cb-4835-ac3c-5931999dc36a.png)

1. En el panel lateral izquierdo, localiza y haz clic en el botón **"Usuarios"**
2. Esta acción abrirá la sección donde podrás ver todos los usuarios disponibles
3. El sistema cargará la lista completa de usuarios registrados

### Paso 2: Elección Del Usuario

![Seleccionando usuario de la lista](https://jam.dev/cdn-cgi/image/width=2560,quality=100,dpr=1.25/https://cdn-jam-screenshots.jam.dev/e40a67fb5b6fd880b791f1d8b28cbf38/screenshot/f7f17ddd-68c3-4da7-8119-df50c97cdbea.png)

1. Se mostrará la lista completa de usuarios disponibles en el sistema
2. Revisa la lista y busca el usuario con el que deseas iniciar una conversación
3. Haz clic sobre el nombre del usuario seleccionado
4. Automáticamente se abrirá una nueva ventana de chat con ese usuario

### Paso 3: Escribir El Mensaje

![Escribiendo mensaje al usuario](https://jam.dev/cdn-cgi/image/width=2560,quality=100,dpr=1.25/https://cdn-jam-screenshots.jam.dev/34d8892152e15808967be4180641cf62/screenshot/d096c80c-8354-42c5-bdc2-073c10c6cc09.png)

1. Una vez abierta la conversación, verás el campo de texto en la parte inferior
2. Escribe tu mensaje en el campo de entrada
3. Opcionalmente, puedes usar las siguientes opciones:
   - **Adjuntar archivos** (icono de clip)
   - **Grabar mensaje de voz** (icono de micrófono)
   - **Insertar emojis** (icono de carita)

### Paso 4: Enviar El Mensaje

![Mensaje enviado correctamente](https://jam.dev/cdn-cgi/image/width=2560,quality=100,dpr=1.25/https://cdn-jam-screenshots.jam.dev/6f086784e579b32e48c9eed3e606f5c7/screenshot/450a0ea7-d09c-41dd-b93f-4c2a77547ee1.png)

1. Haz clic en el botón de **enviar** (icono de avión de papel)
2. El mensaje aparecerá en la conversación del lado derecho (color azul)
3. Verás la hora de envío del mensaje
4. El campo de texto quedará limpio y listo para continuar la conversación

### Paso 5: Chat Guardado En El Panel

![Chat aparece en el panel de chats](https://jam.dev/cdn-cgi/image/width=2560,quality=100,dpr=1.25/https://cdn-jam-screenshots.jam.dev/a6747a786d49027c0dfeadaebeda0600/screenshot/02d05c2b-c783-44a7-a450-35ed75b512d5.png)

1. Una vez iniciada la conversación, el chat con el usuario aparecerá automáticamente en el panel de chats
2. Podrás ver la lista de todas tus conversaciones activas en el panel lateral
3. Cada chat mostrará el nombre del usuario y un extracto del último mensaje
4. Puedes acceder rápidamente a cualquier conversación haciendo clic sobre ella en el panel
