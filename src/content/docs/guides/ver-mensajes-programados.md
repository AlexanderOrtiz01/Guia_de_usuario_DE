## Pasos Para Ver Mensajes Programados

### Paso 1: Configurar Los Filtros De Búsqueda

![Filtros de mensajes programados](https://jam.dev/cdn-cgi/image/width=2560,quality=100,dpr=1.25/https://cdn-jam-screenshots.jam.dev/d54b77abfd8c4e66b04c496765e6180c/screenshot/e58c8f03-f6bb-4657-91fa-b86d5c7d31b2.png)

1. **Tipo**: Selecciona el tipo de notificación que deseas consultar:
   - **Grupales**: Para ver notificaciones enviadas a grupos
   - **Individuales**: Para ver notificaciones enviadas a estudiantes individuales
2. **Estado**: Selecciona **"Programado"** para filtrar solo las notificaciones que están pendientes de envío
3. **Fecha inicio**: Selecciona la fecha y hora desde la cual deseas buscar notificaciones programadas
4. **Fecha fin**: Define la fecha y hora límite del rango de búsqueda
5. Haz clic en el botón **"Actualizar"** para aplicar los filtros y ver los resultados

### Paso 2: Presionar Actualizar

![Botón actualizar](https://jam.dev/cdn-cgi/image/width=2560,quality=100,dpr=1.25/https://cdn-jam-screenshots.jam.dev/6684627a55be59322e4e2e8ccb80e055/screenshot/2f9dafb6-7808-4ea0-b7c9-4e62854a778e.png)

1. Una vez configurados todos los filtros, localiza el botón **"Actualizar"**
2. Haz clic en el botón para ejecutar la búsqueda
3. El sistema procesará los filtros aplicados
4. Los resultados se cargarán en el área de visualización

### Paso 3: Ver Los Resultados

![Resultados de notificaciones programadas](https://jam.dev/cdn-cgi/image/width=2560,quality=100,dpr=1.25/https://cdn-jam-screenshots.jam.dev/8aae7a08d01fb1a3a1a9cd5b816eadb1/screenshot/c6b720c1-afc5-4717-9f97-9a0633d4c73e.png)

1. El sistema mostrará todas las notificaciones programadas que coincidan con los filtros
2. Podrás ver los detalles de cada notificación:
   - Título de la notificación
   - Mensaje programado
   - Fecha y hora programada de envío
   - Estado actual
3. Desde esta vista puedes gestionar las notificaciones programadas según sea necesario
