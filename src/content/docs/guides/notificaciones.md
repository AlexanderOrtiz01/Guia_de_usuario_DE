## Descripción De Elementos

| Numero De Seccion | Descripcion |
|-------------------|-------------|
| 1. Filtro: Tipo | Permite filtrar notificaciones por tipo: Grupales o Individuales. |
| 2. Filtro: Estado | Permite filtrar notificaciones por estado: Programado o Enviado. |
| 3. Filtro: Fecha inicio | Campo para seleccionar la fecha y hora desde la cual buscar notificaciones. |
| 4. Filtro: Fecha fin | Campo para definir el límite final del rango de búsqueda de notificaciones. |
| 5. Botón: Actualizar | Aplica los filtros seleccionados y recarga los resultados de la consulta. |
| 6. Área de resultados | Muestra los resultados de la búsqueda; en este caso, indica "Sin notificaciones". |
